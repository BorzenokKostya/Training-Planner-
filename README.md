# Training Planner

**Автор:** Татьяна Рухтина

## Описание
"Training Planner" — простое приложение для планирования и учёта тренировок с возможностью фильтрации, сохранения и загрузки данных в формате JSON.

## Использование

1. Введите дату, тип тренировки и длительность.
2. Нажмите «Добавить тренировку».
3. Для фильтрации по параметрам заполните фильтр и нажмите «Применить фильтр».
4. Можно сохранить или загрузить ваши тренировки через JSON.
